# Student Progress Tracking System

## Features
- Automatic video watch tracking
- PDF reading engagement tracking
- Quiz completion tracking
- Time spent monitoring
- Module-wise progress dashboard

## Installation & Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and go to:
```
http://localhost:3000
```

## Usage
- Navigate through modules
- Watch videos (progress auto-tracked)
- Read PDF materials (scroll tracking)
- Complete quizzes
- View your progress on the dashboard

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Tailwind CSS
- Backend: Node.js, Express
- Database: JSON files
